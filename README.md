# Expense Tracker (MERN)

**change config.env file in config folder**

## Getting Started.
 > npm install
 > cd client npm install
 > cd ..
 
 ## Run front and backend
 > npm run dev
 
 ## Backend only
 > npm run server
 
 ## Frontend only
 > npm run client
 
 ## Build client
 > cd client
 > npm run build
 
 # For Production - Switch to production branch
 ## Prod
 > npm start
